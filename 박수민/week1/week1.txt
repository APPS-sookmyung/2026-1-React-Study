1.4 변수와 상수
변수, 상수: 값을 저장하는 박스

변수 선언, 변수 초기화(초기값을 넣음) - let 사용
변수는 언제든 값을 바꿀 수 있어서 선언할 때 초기값을 설정하지 않아도 됨
let을 사용한 변수 선언은 이름으로 구별하기 때문에 중복된 이름으로 다시 선언은 어려움
변수는 프로그램 실행 중 계속 바뀔 수 있음

상수는 변수와 달리 한 번 저장된 값을 다시 바꿀수는 없음
-const 사용
값을 변화시키는 것 자체가 불가->초기화가 반드시 필요
선언과 동시에 초기값 할당(초기화)

1.5 자료형 (원시타입 위주)

string타입
number타입

원시타입
-기본형 타입
-프로그래밍에 있어 가장 기본적인 값들의 타입을 의미

1. Number Type 
존재하는 모든 숫자값 포함
정수, 실수, 음수 모두 포함
기본적인 사칙연산 모두 지원

NaN: Not a Number 수치연산에 실패했을 때 결과값

2. String Type
js에서 문자열 값은 " 혹은 '로 해야함
js 문자열은 덧셈 연산 지원
백틱` : 기본적으로는 ""와 동일하지만 변수의 값을 동적으로 넣을 수 있음
->$와 {}를 통해 변수의 값을 동적으로 문자열에 포함시킬 수 있음
->템플릿 리터럴 문법 자주 사용함

3. Boolean Type
true/false
현재 상태를 말할 때 자주 사용


5. undefined type
undefined만 포함하는 특수한 타입
자동으로 할당되는 값

null과 undefined의 차이
undefined는 진짜 변수를 선언하고 아무 값이 안 들어갔을 때 자동으로 들어가는 값
null은 직접 변수에 할당해줘야함
